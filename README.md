# marks.dev

Content for my homepage, [marks.dev](https://marks.dev).

## the basic idea

Create a documentation-as-code based website through Docsify for my domain.

More to come later.

## development

```bash
npm i docsify-cli -g
cd site
docsify serve
```

This allows local preview and live editing of the site.
